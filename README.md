### 🛠 Стек технологій та навички

**Backend & Core:**
* **Мови:** Python (ООП, робота з консоллю)
* **Фреймворки:** Django (проєкти різної складності)
* **Бази даних:** SQL (SQLite, MySQL)
* **Тестування:** Розуміння принципів тестування ПЗ

**Frontend & Mobile:**
* **Веб-технології:** HTML5, CSS3, JavaScript, jQuery, Bootstrap
* **Верстка:** Кросбраузерна та кросплатформна адаптація
* **Mobile:** Створення додатків на базі бібліотеки Kivy

**Інструменти:**
* **Version Control:** Git & GitHub
* **OS:** Досвідчений користувач Windows
